This section contains the complete technical reference for AgentOps. Use it to look up exact field names, API endpoints, and available tools when writing Agent CRs, integrating with the memory service, or selecting tool servers for your agents.

- **[CRD Reference](crds/)** -- full spec for all five Custom Resource Definitions (Agent, AgentRun, AgentTool, AgentResource, Channel).
- **[Memory API](memory-api/)** -- REST endpoints for the agentops-memory service, including context injection, search, and observation management.
- **[Tool Catalog](tool-catalog/)** -- every built-in MCP tool server with tool names, descriptions, and OCI artifact references.
